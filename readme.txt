Desarrollo del trabajo:

Se creó una página de compras para un vivero, a partir de React, utilizando contexto para tomar datos y armar el listado de productos que alimenta al carrito, pudiendo ver las descripciones de los mismos y eligiendo la cantidad deseada a partir del stock disponible.
Se utilizó RouterDOM para hacer posible la navegación entre las distintas páginas(carrito, detalle de productos, contacto e información sobre el emprendimiento).
